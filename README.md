* Checkout docs: [docs/index.md](docs/index.md)

## Getting started

```bash
# download repository
git clone git@github.com:andersonbosa/go-restful-api.git

# enter in the package folder
cd go-restful-api/go-restful-api

# install dependencies
go get .

# run the Gin server
go run .
```
